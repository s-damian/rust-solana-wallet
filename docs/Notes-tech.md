## Networks

| Name         | Type              | Environment      |
|--------------|-------------------|------------------|
| **Devnet**   | Public devnet     | For development  |
| **Testnet**  | Public testnet    | For last testing |
| **Mainnet**  | Production        | For production   |



## Run tests (Localnet)

```bash
cd /<your-path>/rust-solana-wallet
```

```bash
cargo test
```



## Lint

```bash
cd /<your-path>/rust-solana-wallet
```

Format the code:

```bash
cargo fmt
```

Lint:

```bash
cargo clippy
```
